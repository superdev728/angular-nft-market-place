# django-vue-chat
---

# 简介
- 一个用Django和vue搭建的聊天室(websocet使用django-channels实现)

---

# 技术栈
- 前端：vue2.0 全家桶
- 后端：django2.0 django-rest-framewok channels2.0 redis

---
# 安装
- 前端：npm install ->  npm run dev
- 后端：启动redis-server -> python manage.py runserver 0.0.0.0:9090
- 访问：localhost:8080
